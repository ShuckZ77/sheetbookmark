More than one SheetBookmark sheet was found in your Drive. Which one should this browser join?

Sync

The toolbar button always writes immediately. In Manual mode, Ctrl/Cmd + D bookmarks wait for Sync now — the toolbar icon shows how many are queued.

“Get bookmarks” copies what your other browsers saved into a SheetBookmark folder here — it only ever adds, never deletes.

This install

Names this browser's tab in the sheet and is written into every row it saves. No browser exposes its profile or device name to extensions, so describe this install yourself — e.g. Chrome — MacBook Pro.

Rename the sheet, its tabs, or move it anywhere in Drive — sync follows the file, not the name.

Advanced

Reset signs this browser out and clears its local sync record. Your sheet in Drive is left untouched.
