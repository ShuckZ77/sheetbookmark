Bookmark Sheet Sync

Save a bookmark in any browser and read it back in the others. Everything lives in one Google Sheet in your own Drive, with a separate tab for each browser so nothing ever overlaps.

Publisher setup

This build has no Google OAuth client ID, so Connect is disabled. Create one Web application OAuth client, register the redirect URI below, and save the client ID to .keys/client-id.txt, then rebuild. End users never see this card.

Register this on the same OAuth client. Each browser produces a different one, so open this page in every browser you support and add each value.

1 Connect

Connected This browser writes to the tab — in your sheet.

Rename the sheet, its tabs, or move it anywhere in Drive — sync follows the file, not the name.

One click authorizes the extension to create and manage its own bookmark sheet in your Google Drive — nothing else in your Drive is touched. If a sheet from another browser already exists, it is found and reused, never duplicated.

More than one Bookmark Sync sheet was found in your Drive. Which one should this browser join?

2 Sync

Saving with the toolbar button always writes immediately. In Manual mode, bookmarks made with Ctrl/Cmd + D wait for the Sync now button — the toolbar icon shows how many are waiting.

“Get bookmarks” copies anything your other browsers have saved into a Bookmark Sync folder here. It only ever adds — nothing is deleted or overwritten.

3 This install

Names this browser's tab in the sheet and is written into every row it saves. No browser exposes its profile or device name to extensions, so describe this install yourself — e.g. Chrome — MacBook Pro.

Reset

Signs this browser out and clears its local sync record. Your bookmark sheet in Drive is left untouched.
