Features

Built for people who live in more than one browser

Browser sync only syncs a browser with itself. SheetBookmark syncs your bookmarking — across engines, machines and profiles — into a file you can open, sort, filter and keep forever.

A tab per browser

Chrome, Firefox and Edge each write to their own tab in the same spreadsheet — nothing ever overlaps or overwrites. Rename tabs or the sheet freely; sync follows the file, not the name.

Your cadence

Instant by default — a starred page appears in the sheet within seconds. Prefer batches? Every 15 minutes, hourly, daily, or manual-only with a pending-count badge.

Both directions

Read and search every browser's bookmarks from the popup. One click pulls what your other browsers saved into this one — additive only, nothing is ever deleted.

Notes that follow you

Every bookmark in the popup has a ✎ — add or edit its note any time, from any browser. The extension rewrites exactly one cell in your sheet, nothing else.

Feather-light by design

No content scripts, no background polling, zero idle wakeups — the extension does nothing at all until you bookmark something. Your browser won't notice it exists.

Radically private

No server, no telemetry, no account. Google's drive.file scope means the extension can only touch the one spreadsheet it created — enforced by Google, not promised by us.

How it works

Three steps, then forget it exists

Install

Add SheetBookmark to any browser you use — each install is a first-class citizen.

Connect Google Sheets

One click, one consent screen. The extension creates “My SheetBookmark Collection” in your Drive — or finds the one your other browser already made — and imports your existing bookmarks.

Bookmark as you always did

Ctrl/Cmd + D, the star icon, or the toolbar button. Rows appear in that browser's tab; the popup shows the union of all of them.

Your data

What the sheet actually looks like

A stable core of columns (A–I) that never changes, then optional extras appended at the tail. Sort it, filter it, add your own columns after the last one — it's a normal spreadsheet, because it is one.

Privacy

The whole model is “there is nothing to trust”

✓No server. Your browser talks to Google's Sheets & Drive APIs. Nothing passes through us — there is no “us” to pass through.
✓No telemetry. Zero analytics, zero tracking. Even error logs stay on your device — you choose if and when to share them.
✓Minimum scope. drive.file — the extension can only access the spreadsheet it created. The rest of your Drive is invisible to it.
✓Token hygiene. A ~1-hour access token in session memory, never written to disk. No client secret exists anywhere.

FAQ

Questions people actually ask

How do I sync bookmarks between Chrome and Firefox?

Install SheetBookmark in both, click Connect Google Sheets in each with the same Google account. Chrome finds the sheet Firefox created (or vice versa) and adds its own tab. The popup in either browser shows both collections; “Get bookmarks from other browsers” copies them into the local bookmark bar.

Does it work across computers?

Yes — the sheet lives in your Google Drive, so any install connected to the same Google account joins it, on any machine or OS.

What happens if I delete a row, rename the sheet, or move it?

Rename and move anything — sync follows the file's identity, not its name or location. Deleting a row tells the extension that page is no longer saved, so bookmarking it again writes a fresh row.

Can it read my passwords or browsing history?

Neither. Browsers expose no password API to extensions, and SheetBookmark requests no history access. It reads a page's content only at the moment you click Save on it — nothing else, ever.

Something broke — how do I report it?

Options page → Support → Report a problem. It opens a GitHub issue pre-filled with your version, browser and recent on-device error entries (URLs scrubbed) — you see and can edit everything before submitting. Nothing is ever sent automatically.

Is it really free?

Yes. It's open-source, has no server costs to recoup, and Google Sheets is free. There is nothing to upsell.
