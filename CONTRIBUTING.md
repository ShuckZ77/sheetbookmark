# Contributing to SheetBookmark

This project grows by collaboration — improvements belong here, not in private forks.

- **Found a bug?** [Open an issue](https://github.com/ShuckZ77/sheetbookmark/issues/new) —
  the extension's options page → Support → *Report a problem* pre-fills the useful details.
- **Have an idea?** Open an issue first so we can shape it together before you build.
- **Sending a PR?** Keep it small and focused. `npm test` must stay green (62 tests, no
  browser needed), `npm run build` must produce both targets, and new behavior needs a test.
  Style: match the surrounding code; no new dependencies (the zero-dependency build is a feature).
- **Ground rules:** the schema core (columns A–I) is frozen; extensions append at the tail.
  No telemetry, no new permissions without discussion — the privacy posture is the product.

Licensed **AGPL-3.0**: use it, learn from it, build on it — and share your improvements
openly, with attribution. That is the deal that keeps this project worth contributing to.
